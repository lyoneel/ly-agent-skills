# Agent Loader

A skill for loading custom agent definitions and activating them as system prompt overrides.

Originally designed for Windsurf. It lets you define specialized agent personas in Markdown files, then activate them as system prompt overrides for the current session. It can be used in Crush as well, though it was never adapted specifically for it.

Includes one example agent: `GROK-writing-assistant`, a copy of the Grok writing assistant, included purely to demonstrate how agent definitions work.

This skill is kept here as an archived reference.
